# Norway Electricity Demand and Supply Equilibrium

This repository contains a course project analysing electricity demand and
supply equilibrium in Norway using economic theory and empirical data.

## Overview
The project estimates electricity demand and supply curves, identifies the
market equilibrium price and quantity, and illustrates how shifts in demand
or supply affect equilibrium outcomes.

## Methods
- Data cleaning and preparation in Python
- Linear approximation of demand and supply curves
- Analytical and numerical equilibrium solution
- Visualization of equilibrium and comparative statics

## Tools
- Python
- pandas
- numpy
- matplotlib

## Notes
This project is intended as a foundational equilibrium analysis and abstracts
from network constraints and zonal pricing.
